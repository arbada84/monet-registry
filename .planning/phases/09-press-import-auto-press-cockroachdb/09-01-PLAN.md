**싱글톤 Pool:**
```typescript
import { Pool } from "pg";
let pool: Pool | null = null;
export function getPool(): Pool {
  if (!pool) {
    pool = new Pool({
      connectionString: process.env.COCKROACH_DATABASE_URL,
      ssl: { rejectUnauthorized: false },
      max: 5,
      idleTimeoutMillis: 20_000,
      connectionTimeoutMillis: 10_000,
    });
    pool.on("error", () => { pool = null; });
  }
  return pool;
}
```

**PressFeed 인터페이스:**
- id: string (UUID), source: string, source_no: number, title: string, url: string
- date: string|null, category: string|null, company: string|null, summary: string|null
- body_html: string|null, thumbnail: string|null, images: string[], tags: string[]
- crawled_at: string, registered: boolean, article_id: string|null

**CRUD 함수 (모두 파라미터 바인딩 $1, $2... 사용):**
- `getPressFeeds(options: { source?, category?, search?, page?, pageSize? })` — 목록 조회 + COUNT + ORDER BY crawled_at DESC + LIMIT/OFFSET
- `getPressFeedByUrl(url: string)` — 단건 조회 (detail API용)
- `getUnregisteredFeeds(options: { keywords?, dateFrom?, limit? })` — registered=false 미등록 건 (auto-press용)
- `markAsRegistered(feedId: string, articleId: string)` — UPDATE registered=true, article_id=$1

**parsePressFeedRow 내부 함수:**
- images/tags 필드가 string이면 JSON.parse, 아니면 배열 그대로, null이면 빈 배열 반환
- 안전 파싱: try/catch로 JSON.parse 실패 시 빈 배열

**주의사항:**
- 문자열 보간 SQL 절대 금지 — 반드시 $1, $2 파라미터 바인딩
- ssl: true 금지 — 반드시 { rejectUnauthorized: false }
- COCKROACH_DATABASE_URL 없으면 getPool()에서 에러 throw (환경변수 누락 조기 발견)
